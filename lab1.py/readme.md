A simple command-line Python program to help you log your meals and track daily calorie intake. Perfect for anyone looking to monitor their diet and stay within a calorie goal.

Features:
- Interactive prompts to enter meal names and calorie values
- Calculates:
-  Total calories consumed
-  Average calories per meal
- Compares intake against a user-defined daily calorie limit
- Displays a clean summary table
- Optionally saves the log to a file (calorie_log.txt)
- 


- This is a beginner-friendly script ideal for learning basic Python concepts like lists, loops, conditionals.

- You can extend it with features like date-based logging, nutrition breakdowns etc.
